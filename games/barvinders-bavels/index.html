Barvinder's Bavels

a local multiplayer, bomb duel

info

about

Barvinder's Bavels is a small local multiplayer 1 versus 1 game, made in a few class periods. Place down bombs that damage enemy players, the first to zero lives loses. With each playthrough, the game speeds up. Can you keep up ?


controls

Player One

Movement: WASD

Bomb: F


Player Two:

Movement: IJKL

Bomb: ;
